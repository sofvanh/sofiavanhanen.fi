I'm a computer scientist and software engineer building technology for a future full of flourishing. I work primarily in AI R&D, but my stack and interests are vast; I've built collective reasoning platforms, social media apps, multiplayer games; web, mobile, desktop, backend. I view software and AI as tools for uplifting, empowering, liberating, and connecting society.

I'm co-founder and CTO at Mosaic Labs. We build experimental LLM interfaces for AI-mediated collective reasoning and deliberation, conducting research into how AI assistance can help humans coordinate.

I work with many groups on projects that range from AI interfaces to digital democracy and futures studies. Please reach out if you'd like to work with me.

Project highlights

Nexus is a platform for group deliberation that uses large language models to help groups understand each other better and surface insights and solutions that wouldn't have been found otherwise. Link to app, repository, docs

Autoblog is an interface that helps writers publish authentic thoughts while enabling readers to make sense of them in their unique context. Using AI to dynamically translate and personalize content, Autoblog demonstrates how technology can scale nuance and context in ways previously impossible. Link to app and docs, repository

Pantheon is an AI-facilitated metacognition tool exploring new ways to use LLMs to improve human thinking. Intricate, customizable AI characters and a novel, tree-like UI structure empower the user to think and work in new ways. Link to app, repository

Cities of Orare is a vision of a positive future in 2045, where AI-powered prediction markets and digital democracy amplify collective intelligence, enhancing liberal democracy, economic distribution, and policy-making. Cities of Orare won the Foresight Institute Worldbuilding Course of 2024. Link to worldbuild, podcast interview

Harmony was a Slack- and Discord-based chatbot facilitating users in the double crux technique, enabling smoother argument resolution. Discussion, repository, Epistea fellowship
